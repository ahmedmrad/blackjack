## Setup 
To run this game locally: 
1) Clone the project
```sh
$ git clone https://github.com/ahmedmrad/blackjack.git
$ cd blackjack
```
2) Install virtual environement and dependencies

```sh
$ python -m venv env
$ source env/bin/activate
$ pip install -r requirements.txt 
```
3) Play the  game
```sh
$ python run_blackjack.py
```
